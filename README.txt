The goal of this application is to predict the CO2 emissions based on the characteristics of the car. 

Model Pipeline developed with pycaret; a low code ML library.

Data Source:  https://www.kaggle.com/debajyotipodder/co2-emission-by-vehicles/version/1?select=CO2+Emissions_Canada.csv

Link to the application: https://georgios7-co2-ml-app-co2-app-ae48di.streamlit.app/
